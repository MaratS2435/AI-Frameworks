# AI-Frameworks

### Классификация
Ссылка: https://www.kaggle.com/datasets/tan5577/heart-failure-dataset \
Датасет для обучения модели предсказывать сердечную недостаточность по различным признакам пациента. \
Это важно для ранней диагностики и лечения заболевания.

### Регрессия
Ссылка: https://www.kaggle.com/datasets/kundanbedmutha/exam-score-prediction-dataset \
Датасет для обучения модели предсказывать оценки студентов по различным признакам. \
С помощью обученной модели на каких аспектах жизни студенту стоит акцентировать внимание, чтобы улучшить оценки на экзаменах.
